1.什麼是Git？

　　Git 是一種版本控制工具，用來記錄程式碼或文件的修改歷史。就像「存檔系統」，可以回到過去版本，也能和別人一起同時開發專案而不會互相覆蓋。

Q: 如何使用？

　　在電腦安裝 Git ，解壓縮後，用 git init 建立專案，再用 git add、git commit 來記錄檔案的修改。

2.什麼是 Github？

　　GitHub 是一個線上的程式碼託管平台，利用 Git 來管理專案。簡單說： GitHub = 程式碼的「社群網站」，你可以把專案放上去，和其他人合作、分享，甚至找實習或工作時也能當作品集。

Q: 如何使用？

　　到瀏覽器上面註冊 GitHub 帳號，登入以後，建立專案倉庫(repository)，把本地的 Git 用 git push 上傳。

3.什麼是 Gist？

　　Gist 是 GitHub 提供的一個功能，讓你快速分享一小段程式碼或筆記。例如：你寫了一個小範例程式、不想開一個完整專案，就可以用 gist 分享給朋友。

Q: 如何使用？

　　登入 GitHub ，點 Gist 並貼上寫好的程式碼，就可以生成連結分享給他人。

4.什麼是 Cmsimde？

　　Cmsimde 是一個簡易的靜態網站內容管理系統，整合了 Github Pages 與 Markdown 編輯。

Q: 如何使用？

　　從老師提供的 GitHub 倉儲下載 CMSimde 原始檔案，在上面輸入 python main.py 啟動網站，就可以開啟網站。

5.為什麼要使用？

  • Git ：管理檔案版本，能回到過去，不怕檔案遺失。
  • Github ：把專案放到雲端，方便合作與成果展示。
  • Gist ：快速分享一小段程式筆記。
  • CMSimde ：用來快速建立網站，能把專案或學習歷程做成網頁。
